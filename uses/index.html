IDEs

I love VS Code as a lightweight editor, but I use Visual Studio for most of my work. I was hesitant for years to try ReSharper until I watched a friend using it in a legacy codebase, and I had to try it out. It makes refactoring tasks so much faster. I've had various C++ development jobs over the years, so I started using VIM long ago. When there is an option to install a VIM plugin, I will take that option. I'm not a VIM guru, but I know enough to be dangerous.

Desk

I've been using standing desks of some variety for over 10 years. I have used some that are fixed and do not allow automatic adjustment. I've had this desk for about four years, and I love it. I got the bamboo top but didn't get the wheels, and I somewhat regret that.

Jarvis Standing Desk

Chair

I got so lucky when I found this chair at a thrift store for $40. This one has a mesh back and rollerblade wheels which I love way more than I thought I would.

Update: I finally discovered why the chair was so cheap when it started sinking randomly. Shout out to Crandall Office Furniture for a quick replacement part and very helpful instructional videos.

Lights

I have been going a bit crazy over my office lighting. I have several light strips around my desk and my bookcase. LIFX controls have a lot of features, and my favorite feature is the schedule so that I can automatically dim the lights to remind me that it's time for bed.
